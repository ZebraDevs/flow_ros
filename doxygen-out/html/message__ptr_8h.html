◆ NON_ROS_MESSAGE_SHARED_PTR_TMPL

#define NON_ROS_MESSAGE_SHARED_PTR_TMPL   std::shared_ptr<MsgT>

ROS messages are stored as boost::shared_ptr resources, which was a design decision made before C++ had introduce std::shared_ptr with C++11.

This definition provides a fall-back for non-message entities passed between flow_ros Publisher and Subscriber objects through local message passing using flow_ros::Router, since there is no restriction to use boost in these cases.

If you would like to use boost::shared_ptr, or something else, define NON_ROS_MESSAGE_SHARED_PTR_TMPL = shared_ptr before this point
